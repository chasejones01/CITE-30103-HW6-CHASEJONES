# CITE-30103-HW6-CHASEJONES

This repository includes all solutions for Homework #6 of CITE 30103 - Unix/Linux System Administration. Each question shows an important C programming concept, such as variables, operators, typecasting, control statements, loops, and file structure. All .c files compile and independently run. Also, each question includes a screenshot of successful output inside the screenshots/ folder. Each .c file can be compiled and run using the GCC compiler from the terminal.
To compile and run the program:
For Linux/Mac:
    gcc question1.c
    ./a.out

    *NOTE: user can replace question1.c with any other file name to run adn test different programs.
